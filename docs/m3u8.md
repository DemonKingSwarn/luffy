# Quality Selection (m3u8)

After stream decryption, the URL returned by `DecryptStream` is often a **master m3u8 playlist** that lists multiple quality variants rather than a playable stream directly. `core/m3u8.go` fetches that playlist, parses the variants, and either auto-selects the best or shows an fzf menu for the user to choose.

## How It Works

### 1. Fetch and Parse the Master Playlist

`GetQualities` fetches the m3u8 URL, setting the `Referer` header to the value returned by `DecryptStream`. Without this header, CDNs (Megacloud, sflix, braflix) respond with HTTP 403 and quality selection silently degrades to passing the master URL directly to the player.

The function scans the response line by line looking for `#EXT-X-STREAM-INF` tags. Each tag is followed on the next non-comment line by the variant URL:

```
#EXT-X-STREAM-INF:BANDWIDTH=5000000,RESOLUTION=1920x1080
https://cdn.example.com/hls/1080p/index.m3u8
```

For each variant the parser extracts:
- `BANDWIDTH` ‚Äî bits per second, as an integer.
- `RESOLUTION` ‚Äî `{width}x{height}` string; `height` is stored separately for sorting.

Relative variant URLs are resolved against the master playlist's base URL.

### 2. Determine if It's a Master Playlist

If no `#EXT-X-STREAM-INF` line is found the file is a **media playlist** (a single-quality stream). `GetQualities` returns `(nil, originalURL, nil)` ‚Äî the caller should use the original URL directly.

If variants are found, the parsed `[]Quality` slice is returned and the direct URL return value is `""`.

### 3. Select a Quality

`SelectQuality` decides how to pick from the parsed variants:

| Condition | Behaviour |
|-----------|-----------|
| `selectBest == true` | Calls `GetBestQuality`, returns the URL, no prompt |
| Single variant | Calls `GetBestQuality` automatically, no prompt |
| Multiple variants, `selectBest == false` | Launches fzf with variant labels |

Labels shown in fzf are the `Resolution` string (e.g. `1920x1080`) when available, or a human-readable bandwidth string when resolution is absent (e.g. `4.8 Mbps` or `720 Kbps`).

### 4. Best Quality Selection

`GetBestQuality` iterates the variants and picks the one with the highest `Height`. Ties are broken by `Bandwidth`. This is a pure sort ‚Äî the fzf prompt is never shown.

### 5. Integration in cmd/root.go

The caller pattern after decryption:

```go
if strings.Contains(streamURL, ".m3u8") {
    qualities, directURL, err := core.GetQualities(streamURL, ctx.Client, referer)
    if err == nil && len(qualities) > 0 {
        selectBest := bestFlag || strings.EqualFold(cfg.Quality, "best")
        streamURL, err = core.SelectQuality(qualities, selectBest)
    } else if directURL != "" {
        streamURL = directURL
    }
}
```

`referer` is set from the third return value of `DecryptStream`. For sflix and braflix it is overridden with the embedder's origin:

```go
if parsedURL, err := url.Parse(link); err == nil {
    referer = fmt.Sprintf("%s://%s/", parsedURL.Scheme, parsedURL.Host)
}
```

## Key Types

```go
// Quality represents one quality variant from a master m3u8 playlist.
type Quality struct {
    URL        string // absolute URL of the variant playlist
    Resolution string // e.g. "1920x1080"; empty if not in playlist
    Bandwidth  int    // bits per second
    Height     int    // vertical pixels; 0 if RESOLUTION was absent
    Label      string // text shown in fzf (Resolution or bandwidth string)
}
```

## Public API

```go
// GetQualities fetches the m3u8 at m3u8URL (passing referer as the
// Referer header) and returns all quality variants found.
// If the URL is not a master playlist, returns (nil, m3u8URL, nil).
func GetQualities(m3u8URL string, client *http.Client, referer string) ([]Quality, string, error)

// GetBestQuality returns the variant with the highest Height (then
// Bandwidth as tiebreaker). Returns the zero Quality on empty input.
func GetBestQuality(qualities []Quality) Quality

// SelectQuality returns the URL of the chosen variant.
// If selectBest is true (or there is only one variant), auto-picks
// the best without showing fzf. Otherwise shows an fzf selection menu.
func SelectQuality(qualities []Quality, selectBest bool) (string, error)
```
